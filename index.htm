1. Buttons

Buttons can have 5 (five) different accents:

  1. Normal Button
  2. Main Button (silky-main) Used for default buttons.
  3. Success Button (silky-success) Used for accomplishment buttons.
  4. Warning Button (silky-warn) Used for non-recommended action buttons.
  5. Danger Button (silky-danger) Used for dangerous action buttons.

Buttons can also have 3 (three) different sizes:

  1. Large (silky-giant)
  2. Normal
  3. Small (silky-little)
